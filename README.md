# 薪水計算器 salary-calculator

這是一個純前端薪水計算小工具，可以直接放 GitHub Pages 使用。

## 功能

- 年份設定
- 時薪設定
- 自動產生每 14 天一次的發薪表
- 每日工時填寫
- 空白自動當 0
- 自動儲存在瀏覽器 localStorage
- 發薪總覽
- 年度統計
- 深色 / 亮色模式
- 匯出 JSON 備份
- 匯入 JSON 還原
- 清空工時紀錄

## 預設規則

目前預設：

- 時薪：205
- 第一個已知發薪日：2026-04-11
- 發薪日往前 28 天 = 該期結束日
- 該期結束日往前 13 天 = 該期開始日

例：

- 2026/4/11 領 2026/3/1～2026/3/14
- 2026/4/25 領 2026/3/15～2026/3/28
- 2026/5/9 領 2026/3/29～2026/4/11

## 使用方式

直接打開 `index.html` 即可。

如果要放 GitHub Pages，把整個資料夾上傳到 GitHub repository，Pages 設定來源為 main branch/root 即可。
