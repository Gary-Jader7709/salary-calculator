# 薪水計算器 salary-calculator

純前端薪水計算小工具，可直接放 GitHub Pages 使用。

## 功能

- 年份設定
- 時薪設定
- 自動產生發薪表
- 每日工時填寫
- 空白自動當 0
- 自動儲存在瀏覽器 localStorage
- 發薪總覽
- 年度統計
- 深色 / 亮色模式
- 匯出 JSON 備份
- 匯入 JSON 還原
- 清空工時紀錄

## 正式薪水規則

- 上個月 1～15 號 → 這個月第 2 個星期六領
- 上個月 16～月底 → 這個月第 4 個星期六領

例：

- 2026/4/11 領 2026/3/1～2026/3/15
- 2026/4/25 領 2026/3/16～2026/3/31
- 2026/5/9 領 2026/4/1～2026/4/15
- 2026/5/23 領 2026/4/16～2026/4/30

## 使用方式

直接打開 `index.html` 即可。

如果要放 GitHub Pages，把 `index.html`、`style.css`、`script.js`、`README.md` 放在 repo 最外層，Pages 設定 main branch / root。
